# Estadística Descriptiva

Repositorio que contiene material de estudio, ejercicios resueltos y recursos de referencia para estadística descriptiva utilizando Python y pandas.

## Descripción

Este repositorio incluye una colección completa de laboratorios prácticos, datasets y guías de referencia para el aprendizaje y aplicación de conceptos fundamentales de estadística descriptiva. Todos los ejercicios están implementados en Jupyter Notebooks con soluciones completas.

## Estructura del Repositorio

```
Estadistica_Descriptiva/
├── Ejercicios/          # Laboratorios prácticos resueltos
├── Cheat_Sheet/         # Guías de referencia y resúmenes
├── Datasets/            # Bases de datos en formato Excel
└── README.md
```

## Contenido

### Ejercicios

El directorio `Ejercicios/` contiene los siguientes laboratorios:

- **Laboratorio 0**: Introducción a Pandas
- **Laboratorio 1**: Conceptos básicos de estadística
- **Laboratorio 2**: Tablas de frecuencia
- **Laboratorio 3**: Gráficos
- **Laboratorio 4**: Medidas de Tendencia Central y Percentiles
- **Laboratorio 5**: Medidas de Dispersión
- **Laboratorio 6**: Modelos de Regresión Lineal
- **Laboratorio 7**: Probabilidades
- **Laboratorio 8**: Distribución Normal

Cada laboratorio incluye ejercicios prácticos resueltos con explicaciones detalladas y código Python ejecutable.

### Cheat Sheet

El directorio `Cheat_Sheet/` contiene recursos de referencia rápida:

- **interpretaciones.md**: Guía completa de interpretaciones estadísticas, conceptos clave y ejemplos prácticos
- **one.ipynb** y **two.ipynb**: Notebooks de referencia con ejemplos y fórmulas

### Datasets

El directorio `Datasets/` contiene múltiples bases de datos en formato Excel utilizadas en los laboratorios, incluyendo:

- Datos para análisis de dispersión
- Datos para distribución de probabilidad
- Datos para gráficos y visualizaciones
- Datos para medidas de posición
- Datos para modelos de regresión
- Datos para tablas de frecuencias
- Y otros datasets complementarios

## Requisitos

Para ejecutar los notebooks de este repositorio, se recomienda tener instalado:

- Python 3.x
- Jupyter Notebook o JupyterLab
- pandas
- numpy
- matplotlib
- seaborn (para algunas visualizaciones)

## Uso

1. Clona o descarga este repositorio
2. Abre los notebooks en Jupyter Notebook o JupyterLab
3. Ejecuta las celdas en orden para seguir los ejercicios
4. Consulta `Cheat_Sheet/interpretaciones.md` como referencia rápida
